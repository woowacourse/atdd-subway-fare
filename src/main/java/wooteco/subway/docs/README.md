## 요금

- 기본운임(10㎞ 이내) : 기본운임 1,250원
- 이용 거리초과 시 추가운임 부과
    - 10km초과∼50km까지(5km마다 100원)
    - 50km초과 시 (8km마다 100원)

- 실제 경로가 아닌 최단거리 기준(지하철 운임은 거리비례제로 책정)

```
9km = 1250원
12km = 10km + 2km = 1350원
16km = 10km + 6km = 1450원
```

## 5km 마다 100원 추가

```
private int calculateOverFare(int distance) {
    return (int) ((Math.ceil((distance - 1) / 5) + 1) * 100);
}
```
